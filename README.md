# 🎬 Movie Ticket Booking System

A simple **web-based Movie Ticket Booking application** developed using **HTML, CSS, and JavaScript**.
This project allows users to browse movies by genre, choose language and location, select a theatre, view available movies, and simulate booking a ticket.

---

## 🚀 Features

* 🎭 Select movie **genre** (Romantic, Action, Comedy, Horror)
* 🌐 Choose preferred **language**
* 📍 Select **city and theatre**
* 🎬 View available **movies**
* ⏰ Choose **show timings**
* 📝 Enter booking details
* 💳 Simulated **payment process**
* ✅ **Booking confirmation** page

---

## 🛠️ Technologies Used

* **HTML5** – Structure of web pages
* **CSS3** – Styling and layout
* **JavaScript** – Dynamic functionality and page navigation
